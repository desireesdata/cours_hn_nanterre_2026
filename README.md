# UE "Humanités numériques"

Ressources du cours.
Les slides de chaque séance sont dans :

```
hn_seance_[numéro de la séance]/typslides/main.pdf
```

Les lectures à faire sont dans le dossier lectures. La date de la lecture est indiqué dans le nom du fichier.

Exemple : pour le 27/01, le fichier s'appelle :

```
27_01_julien-saavedra-2015-manifeste-pour-des-humanites-numeriques-20
```

## Commandes Typst

```shell
cd blabla
typst watch main.typ
```
